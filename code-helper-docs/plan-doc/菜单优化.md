- 将一级菜单主功能名统一为“功能管理”。
- 功能管理中的 Skills 应用和取消支持按 Codex、Claude Code、GitHub Copilot、全部或当前项目默认目标执行。
- 功能管理中的 Agent hooks 应用和取消支持按 Codex、Claude Code 或全部可用 Agent hooks 执行，明确排除 GitHub Copilot。
- 不回滚他人改动，不提交、不推送、不发布 npm。
